Tabelas

As tabelas são feitas da seguinte forma:

  • Thead: é usado para definir o cabeçalho.
  • Tr ou Th: é usado para definir linhas.
  • Td: é usado para definir colunas.
  • Tfoot: é usado para definir o roda pé.
